# Structured Pipelines

Structured Pipelines provide a sequential set of data processing steps, often used in machine learning workflows.

**Official Documentation / Website**: [https://scikit-learn.org/stable/modules/compose.html](https://scikit-learn.org/stable/modules/compose.html)

## Learning Path

1. Define preprocessing steps.
2. Chain transformations and estimators.
3. Evaluate the full structured pipeline.
