# Tabular Data

Tabular Data is data that is structured into rows and columns, typically found in relational databases or CSV files.

**Official Documentation / Website**: [https://pandas.pydata.org/docs/](https://pandas.pydata.org/docs/)

## Learning Path

1. Learn to read tabular data.
2. Perform data cleaning and handling missing values.
3. Manipulate and aggregate tabular data.
