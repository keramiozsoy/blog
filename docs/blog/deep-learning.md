# Deep Learning

Deep Learning is a subset of machine learning based on artificial neural networks with multiple layers.

**Official Documentation / Website**: [https://www.deeplearning.ai/](https://www.deeplearning.ai/)

## Learning Path

1. Understand basic Neural Networks.
2. Learn about Backpropagation and Optimization.
3. Study Architectures (CNNs, RNNs, Transformers).
