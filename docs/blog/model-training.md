# Model Training

Model Training is the process of feeding data into a machine learning algorithm to help it identify patterns and make predictions.

**Official Documentation / Website**: [https://scikit-learn.org/stable/tutorial/basic/tutorial.html](https://scikit-learn.org/stable/tutorial/basic/tutorial.html)

## Learning Path

1. Split data into training and validation sets.
2. Select an algorithm and train the model.
3. Optimize hyperparameters.
