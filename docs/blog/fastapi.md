# FastAPI

FastAPI is a modern, fast (high-performance), web framework for building APIs with Python based on standard Python type hints.

**Official Documentation / Website**: [https://fastapi.tiangolo.com/](https://fastapi.tiangolo.com/)

## Learning Path

1. Create a basic API endpoint.
2. Implement path and query parameters.
3. Use Pydantic for data validation.
