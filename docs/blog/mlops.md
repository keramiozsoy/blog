# MLOps

MLOps (Machine Learning Operations) is a set of practices that aims to deploy and maintain machine learning models in production reliably and efficiently.

**Official Documentation / Website**: [https://ml-ops.org/](https://ml-ops.org/)

## Learning Path

1. Understand the MLOps lifecycle.
2. Implement CI/CD for machine learning.
3. Set up model monitoring.
