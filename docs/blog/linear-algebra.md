# Linear Algebra

Linear Algebra is the mathematical foundation of machine learning, focusing on vector spaces and linear mappings.

**Official Documentation / Website**: [https://ocw.mit.edu/courses/mathematics/18-06-linear-algebra-spring-2010/](https://ocw.mit.edu/courses/mathematics/18-06-linear-algebra-spring-2010/)

## Learning Path

1. Understand Vectors and Matrices.
2. Learn about Matrix Multiplication and Transformations.
3. Study Eigenvalues and Eigenvectors.
