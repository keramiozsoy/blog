# Building on top of LLMs

Building on top of LLMs involves integrating large language models into software applications to create intelligent features.

**Official Documentation / Website**: [https://python.langchain.com/docs/get_started/introduction](https://python.langchain.com/docs/get_started/introduction)

## Learning Path

1. Select an appropriate LLM provider/model.
2. Design the interaction logic (API calls).
3. Implement error handling and retries.
