# LightGBM

LightGBM is a gradient boosting framework that uses tree based learning algorithms, designed for distributed and efficient training.

**Official Documentation / Website**: [https://lightgbm.readthedocs.io/](https://lightgbm.readthedocs.io/)

## Learning Path

1. Install LightGBM.
2. Prepare data using Dataset.
3. Train, tune, and evaluate LightGBM models.
