# Retrieval Pipeline

The Retrieval Pipeline is the subsystem responsible for searching and returning the most relevant information given a query.

**Official Documentation / Website**: [https://docs.llamaindex.ai/en/stable/](https://docs.llamaindex.ai/en/stable/)

## Learning Path

1. Configure the vector store retriever.
2. Implement hybrid search (keyword + vector).
3. Add reranking to improve relevance.
