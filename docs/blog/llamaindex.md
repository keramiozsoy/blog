# LlamaIndex

LlamaIndex is a data framework for LLM-based applications to ingest, structure, and access private or domain-specific data.

**Official Documentation / Website**: [https://docs.llamaindex.ai/en/stable/](https://docs.llamaindex.ai/en/stable/)

## Learning Path

1. Load data using Document Loaders.
2. Create indices from documents.
3. Query the data using Query Engines.
