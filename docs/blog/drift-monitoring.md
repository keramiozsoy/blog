# Drift Monitoring

Drift Monitoring involves tracking changes in data distribution or model performance over time to detect degradation.

**Official Documentation / Website**: [https://evidentlyai.com/docs](https://evidentlyai.com/docs)

## Learning Path

1. Monitor data drift (input features).
2. Monitor concept drift (target variable).
3. Set up alerts for drift detection.
