# Feature Engineering

Feature Engineering is the process of using domain knowledge to extract features from raw data via data mining techniques.

**Official Documentation / Website**: [https://scikit-learn.org/stable/modules/preprocessing.html](https://scikit-learn.org/stable/modules/preprocessing.html)

## Learning Path

1. Handle missing values.
2. Encode categorical variables.
3. Create polynomial and interaction features.
