# Guardrails

Guardrails are mechanisms to ensure that AI applications operate within defined safety, ethical, and operational boundaries.

**Official Documentation / Website**: [https://docs.guardrailsai.com/](https://docs.guardrailsai.com/)

## Learning Path

1. Define rules and safety constraints.
2. Implement input validation (prompt injection checks).
3. Implement output validation (toxicity, hallucinations).
