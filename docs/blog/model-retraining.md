# Model Retraining

Model Retraining is the practice of periodically training an existing model on new data to maintain its performance over time.

**Official Documentation / Website**: [https://evidentlyai.com/blog/machine-learning-monitoring](https://evidentlyai.com/blog/machine-learning-monitoring)

## Learning Path

1. Set up data collection pipelines.
2. Define retraining triggers (time-based or performance-based).
3. Automate the training and deployment pipeline.
