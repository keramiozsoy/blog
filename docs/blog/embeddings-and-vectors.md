# Embeddings and Vectors

Embeddings are dense numerical vector representations of data (like text) that capture semantic meaning.

**Official Documentation / Website**: [https://platform.openai.com/docs/guides/embeddings](https://platform.openai.com/docs/guides/embeddings)

## Learning Path

1. Understand what embeddings are.
2. Generate embeddings using APIs.
3. Calculate similarity between vectors (Cosine Similarity).
