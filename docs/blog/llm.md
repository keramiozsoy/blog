# Large Language Models (LLM)

Large Language Models are deep learning algorithms that can recognize, summarize, translate, predict and generate text.

**Official Documentation / Website**: [https://platform.openai.com/docs/](https://platform.openai.com/docs/)

## Learning Path

1. Understand the Transformer architecture.
2. Learn about pre-training and fine-tuning.
3. Explore prompting and generation strategies.
