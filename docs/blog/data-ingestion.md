# Data Ingestion

Data Ingestion is the process of moving data from various sources into a storage medium where it can be accessed, used, and analyzed.

**Official Documentation / Website**: [https://airflow.apache.org/](https://airflow.apache.org/)

## Learning Path

1. Identify data sources.
2. Choose ingestion method (batch vs. streaming).
3. Implement data extraction and loading.
