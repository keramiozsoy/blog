# Prompt Engineering

Prompt Engineering is the practice of designing inputs for generative AI tools that will produce optimal outputs.

**Official Documentation / Website**: [https://www.promptingguide.ai/](https://www.promptingguide.ai/)

## Learning Path

1. Learn basic prompt structures.
2. Master techniques like Few-Shot and Chain-of-Thought.
3. Implement dynamic context generation.
