# Supabase

Supabase is an open source Firebase alternative, providing a Postgres database, Authentication, instant APIs, Edge Functions, Realtime subscriptions, and Storage.

**Official Documentation / Website**: [https://supabase.com/docs](https://supabase.com/docs)

## Learning Path

1. Set up a project and database.
2. Implement authentication.
3. Use pgvector for embedding storage.
