# Deploying to GitHub Pages

GitHub Pages is a great tool to share your blog with the world.

## Automatic Deployment
Using GitHub Actions, you can ensure your site is updated automatically every time you `git push`.

### Steps:
1. Upload your repository to GitHub.
2. Add the necessary file under `.github/workflows`.
3. Sit back and share your code!
