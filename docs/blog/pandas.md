# Pandas

Pandas is a fast, powerful, flexible, and easy-to-use open source data analysis and manipulation tool.

**Official Documentation / Website**: [https://pandas.pydata.org/](https://pandas.pydata.org/)

## Learning Path

1. Learn basic Series and DataFrame structures.
2. Master data indexing, selection, and filtering.
3. Perform data aggregation and grouping.
