# NumPy

NumPy is the fundamental package for scientific computing with Python, providing support for large, multi-dimensional arrays and matrices.

## Getting Started
To use NumPy, you first need to install it via pip:
```bash
pip install numpy
```

Then, you can import it and create a basic array:
```python
import numpy as np

# Create a simple 1D array
arr = np.array([1, 2, 3, 4, 5])
print(arr)
```

Official Website: [numpy.org](https://numpy.org/)
