# Vector Database

A Vector Database is a specialized database designed to store, manage, and search for embedding vectors efficiently.

**Official Documentation / Website**: [https://milvus.io/docs](https://milvus.io/docs)

## Learning Path

1. Understand vector indexing algorithms (HNSW).
2. Insert embeddings into the database.
3. Perform similarity searches.
