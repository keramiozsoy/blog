# Seaborn

Seaborn is a Python data visualization library based on matplotlib that provides a high-level interface for drawing attractive statistical graphics.

**Official Documentation / Website**: [https://seaborn.pydata.org/](https://seaborn.pydata.org/)

## Learning Path

1. Create statistical plots (box plots, violin plots).
2. Visualize distributions and relationships.
3. Customize themes and color palettes.
