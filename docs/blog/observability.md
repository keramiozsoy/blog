# Observability

Observability in AI refers to the deep visibility into the internal state of machine learning systems to understand their behavior and performance.

**Official Documentation / Website**: [https://arize.com/blog/what-is-ml-observability/](https://arize.com/blog/what-is-ml-observability/)

## Learning Path

1. Instrument applications to collect logs, metrics, and traces.
2. Set up dashboards for visualization.
3. Implement tracing for complex AI workflows.
