# Model Evaluation

Model Evaluation is the process of assessing a machine learning model's performance on a test dataset.

**Official Documentation / Website**: [https://scikit-learn.org/stable/modules/model_evaluation.html](https://scikit-learn.org/stable/modules/model_evaluation.html)

## Learning Path

1. Choose appropriate evaluation metrics.
2. Evaluate model using cross-validation.
3. Analyze confusion matrix and ROC curves.
