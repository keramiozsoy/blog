# Experiment Design

Experiment Design is the careful planning of studies to ensure that the data collected is suitable to answer the research questions objectively.

**Official Documentation / Website**: [https://www.stat.yale.edu/Courses/1997-98/101/expdes.htm](https://www.stat.yale.edu/Courses/1997-98/101/expdes.htm)

## Learning Path

1. Formulate clear hypotheses.
2. Determine sample sizes and randomization methods.
3. Define control and treatment groups.
