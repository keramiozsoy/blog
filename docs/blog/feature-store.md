# Feature Store

A Feature Store is a centralized repository that enables data scientists to find, share, and use features for machine learning models.

**Official Documentation / Website**: [https://feast.dev/](https://feast.dev/)

## Learning Path

1. Define feature views.
2. Register features in the store.
3. Serve features for training and inference.
