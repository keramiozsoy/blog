# XGBoost

XGBoost is an optimized distributed gradient boosting library designed to be highly efficient, flexible, and portable.

**Official Documentation / Website**: [https://xgboost.readthedocs.io/](https://xgboost.readthedocs.io/)

## Learning Path

1. Install XGBoost.
2. Prepare data using DMatrix.
3. Train and tune XGBoost models.
