# Unstructured Text

Unstructured Text refers to data that does not have a predefined data model or is not organized in a pre-defined manner.

**Official Documentation / Website**: [https://www.nltk.org/](https://www.nltk.org/)

## Learning Path

1. Learn basic text processing (tokenization).
2. Understand techniques like stemming and lemmatization.
3. Extract features from text (TF-IDF, Embeddings).
