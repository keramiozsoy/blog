# PyTorch

PyTorch is an open-source machine learning framework for research and production.

**Official Website:** [pytorch.org](https://pytorch.org/)

## 🎓 Official Learning Path
Follow the official "Learn the Basics" path:

1. [Learn the Basics (Tutorial)](https://pytorch.org/tutorials/beginner/basics/intro.html)
2. [PyTorch Recipes](https://pytorch.org/tutorials/recipes/recipes_index.html)
3. [Deep Learning with PyTorch: A 60 Minute Blitz](https://pytorch.org/tutorials/beginner/deep_learning_60min_blitz.html)

*Note: While there is no official PyTorch certificate, completing these tutorials is recognized as the standard entry point for DL research.*
