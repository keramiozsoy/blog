# CrewAI

CrewAI is a framework for orchestrating role-playing, autonomous AI agents.

**Official Documentation / Website**: [https://docs.crewai.com/](https://docs.crewai.com/)

## Learning Path

1. Define agents with specific roles and goals.
2. Create tasks for the agents.
3. Form a crew and orchestrate task execution.
