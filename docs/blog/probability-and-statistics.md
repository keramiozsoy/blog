# Probability and Statistics

Probability and Statistics provide the tools to understand data distributions, uncertainty, and statistical significance.

**Official Documentation / Website**: [https://www.khanacademy.org/math/statistics-probability](https://www.khanacademy.org/math/statistics-probability)

## Learning Path

1. Understand basic Probability concepts.
2. Learn about Distributions (Normal, Binomial, etc.).
3. Study Hypothesis Testing and p-values.
