# Agent Building

Agent Building involves creating AI systems that can use tools, reason, and take actions to achieve specific goals.

**Official Documentation / Website**: [https://python.langchain.com/docs/modules/agents/](https://python.langchain.com/docs/modules/agents/)

## Learning Path

1. Define agent tools and capabilities.
2. Select the reasoning engine (ReAct).
3. Implement the agent execution loop.
