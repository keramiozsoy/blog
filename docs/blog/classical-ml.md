# Classical ML

Classical ML refers to traditional statistical and algorithmic approaches to learning from data without deep neural networks.

**Official Documentation / Website**: [https://scikit-learn.org/](https://scikit-learn.org/)

## Learning Path

1. Learn about supervised learning algorithms.
2. Understand unsupervised learning and clustering.
3. Evaluate models using appropriate metrics.
