# Keras

Keras is an API designed for human beings, not machines, for building deep learning models.

**Official Website:** [keras.io](https://keras.io/)

## 🎓 Official Learning Path
Follow these guides to learn Deep Learning with Keras:

1. [Keras for Beginners](https://keras.io/getting_started/)
2. [Developer Guides](https://keras.io/guides/)
3. [Keras Code Examples](https://keras.io/examples/)
