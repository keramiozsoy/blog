# Streamlit

Streamlit is an open-source app framework for Machine Learning and Data Science teams to create beautiful data apps in hours, not weeks.

**Official Documentation / Website**: [https://docs.streamlit.io/](https://docs.streamlit.io/)

## Learning Path

1. Install Streamlit and run a basic app.
2. Add interactive widgets (buttons, sliders).
3. Display data and plots.
