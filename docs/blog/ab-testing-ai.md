# A/B Testing for AI Models

A/B Testing for AI Models is a randomized experiment with two variants, A and B, to compare the performance of different models in production.

**Official Documentation / Website**: [https://vwo.com/ab-testing/](https://vwo.com/ab-testing/)

## Learning Path

1. Define the key performance indicators (KPIs).
2. Route traffic between model variants.
3. Analyze results for statistical significance.
