# Eval Suite

An Eval Suite is a collection of tests and metrics used to systematically evaluate the performance of AI models and applications.

**Official Documentation / Website**: [https://docs.smith.langchain.com/](https://docs.smith.langchain.com/)

## Learning Path

1. Define evaluation metrics (accuracy, relevance, groundedness).
2. Create test datasets.
3. Automate the evaluation pipeline.
