# LangGraph

LangGraph is an extension to LangChain for building stateful, multi-actor applications with LLMs using graph structures.

**Official Documentation / Website**: [https://python.langchain.com/docs/langgraph](https://python.langchain.com/docs/langgraph)

## Learning Path

1. Understand the concepts of nodes and edges.
2. Define application state.
3. Build cyclic agent workflows.
