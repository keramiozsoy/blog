# RAG Design

Retrieval-Augmented Generation (RAG) is an AI framework for improving the quality of LLM responses by grounding the model on external sources of knowledge.

**Official Documentation / Website**: [https://python.langchain.com/docs/use_cases/question_answering/](https://python.langchain.com/docs/use_cases/question_answering/)

## Learning Path

1. Design the data ingestion and indexing pipeline.
2. Implement the retrieval mechanism.
3. Combine retrieved context with the prompt for generation.
